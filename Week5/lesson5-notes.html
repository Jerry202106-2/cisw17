Lesson 9: Working Margins, Padding, Alignment, and Floating

Summary:

In this lesson, I learned some of the most fundamental style properties in CSS-based design: margin, padding and float. I learned that the margin controls the space around the outside of elements. And the padding works with space within the elments.

In this lesson, I also learned about the float property. There are left, right in float property. The float property allows for specific placement of elements and additional content arount those elements.

What You'll Learn in This Lesson

  • How to add margins around elements
  • How to add padding within elements
  • How to keep everything aligned
  • How to use the float property

Lesson 10: Understanding the CSS Box Model and Positioning

Summary:

In this lesson, we began with an important discussion about the CSS box model. Like how to calculate the width and height of elements when considering margins, padding, and borders. I also learned change the default box model by using the box-sizing property. I also learned about the positioning using z-index.

I learned about the type of positioning: relative - The element is positioned relative to its current position in the document flow; absolute - The element is positioned based on its container element. fixed - The element is positioned relative to the brow window; static - The element is placed as it appears in the normal flow. This is the default.

What You'll Learn in This Lesson

  • How to conceptualize the CSS box model
  • How to change the box model the browser uses
  • How to position your elements
  • How to control the way elements stack up
  • How to manage the flow of text
