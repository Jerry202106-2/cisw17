Week 6 Lesson Notes

Lesson 12: Creating Layouts Using Modern CSS Techniques

Summary:

In this lesson, I learned about the three main types of layouts: fixed, liquid, and a fixed/liquid hybird. In the middle of the lesson, there is a example of creating a fixed/liquid hybrid layout. It is very helpful to create a wonderful and beautiful layout.

I also learned how create a similar layout by using modern CSS layout with flexiable boxes and grids. I also learn CSS grid, CSS flexible layouts. It is very important for me to create a HTML layout in future.

What You'll Learn in This Lesson

  • How fixed layouts work
  • How liquid layouts work
  • How to create a fixed/liquid hybrid layout
  • How to create CSS flexible box layouts
  • How to use CSS grid layouts
