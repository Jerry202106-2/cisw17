Lesson 14: Using CSS Transformations and Transitions

Summary:

In this lesson, I learnded how to add basic animation to web pages by transforming elements. In the first section of the lesson, I learned how to use "transform" property. I also learned "rotate()" to rotate an element. And I also learned use "scale()" function to make an element larger or smaller. With the "translate()", "skew()", we can move elements from where they would normally be placed and till elements on the horizontal and vertical axes. I also learned use "rotatex()","rotatey()","rotate3d(x,y,z,angle)" to spin an element around a point in space.

I also learned how to animate transformations with the "transition" properties. I learned how to define the properties to transition, how to define the time a transition takes, how to set a delay for a transition, how to define the time a transition takes, and how to set a delay for a transition so that is does not start immediately.

What You'll Learn in This Lesson

  • How to transform elements by rotating, scaling, moving, and tilting
  • How to wok with transformations in three dimensions
  • How to apply multiple transformations to one element
  • How to do simple animations with the transition property
